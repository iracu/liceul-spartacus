# liceul-spartacus
Proiect de curs.

To run this project you need to have node js in your PC.

https://nodejs.org/en/download/

To run the project do next steps:
  1) "cd api/" - folder
    - run command "npm i"
    - run command "npm run start"
  2) "cd client" - folder
    - run command "npm i"
    - run command "npm run start"
  3) After these actions navigate to "http://localhost:3000/"
